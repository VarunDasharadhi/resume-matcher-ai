# 🧠 AI Resume Matcher

A smart, production-ready web app powered by OpenAI's GPT-3.5 to analyze and improve resumes against any job description.

🔍 Upload your resume + paste job description → get:

- ✅ Match Score (0–100)
- ✅ Matching & Missing Skills
- ✅ GPT-powered Resume Suggestions
- ✅ Downloadable Match Report (PDF)
- ✅ AI-Generated Cover Letter (also downloadable)

---

🚀 **Live Demo**: [Try it here](https://ai-dispatch-assistant.onrender.com)

🧰 **Tech Stack**:
- Python · Flask · OpenAI GPT-3.5 · PyMuPDF · ReportLab
- Hosted on Render.com · Fully serverless deployment
